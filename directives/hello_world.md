# Hello World Directive
This is a sample directive to demonstrate the 3-layer architecture.

## Goal
Print "Hello from the Execution Layer!" to the console using a deterministic Python script.

## Inputs
- None

## Tools/Scripts
- `execution/hello_world.py`
- `execution/hello_world.js` (Node.js fallback)

## Expected Output
- A message printed to the console.

## Edge Cases
- Python not installed (unlikely in this environment).
